## [0.1.0] - 2024-11-30

### Added
- Initial release
- FastAPI backend with ESPN F1 API integration
- FastF1 library integration for telemetry data
- SQLite database for caching
- Astro + React frontend
- Race-by-race standings tables
- Driver and Constructor standings
- Sprint race indicators
- Historically accurate points systems (1950-2025)
- Team logos integration
- Country flags for race locations

### Backend Features
- ESPN API client for historical data
- FastF1 wrapper for telemetry
- RESTful API endpoints
- Database population scripts
- Automatic caching system

### Frontend Features
- Interactive standings tables
- Race-by-race results view
- Responsive design with Tailwind CSS v4
- Data visualization with Recharts
- Sprint race indicators and legends

### Database
- Complete schema for F1 data
- Season data from 1950-present
- Race results and standings
- Driver and constructor information
- Points system accuracy for all eras
